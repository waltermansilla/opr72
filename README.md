# OPR72 — Sitio institucional

Landing de OPR72 (Organización de Protección Reconocida Nº 72). Next.js + Tailwind + TypeScript.

**Producción:** [opr72.vercel.app](https://opr72.vercel.app)

## Desarrollo

```bash
npm install
npm run dev
```

Abrí **http://127.0.0.1:3001** (puerto fijo).
